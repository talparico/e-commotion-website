Was ist UX/UI Design?

User Experience (UX) beschreibt wie sich die Nutzung Deines Shops anfühlt. User Interface (UI) ist das visuelle Design. Beides zusammen entscheidet ob Besucher kaufen oder abspringen.

phone_iphone

Warum Mobile-First entscheidend ist

Über 70% aller Shopbesuche kommen vom Smartphone. Ein Shop der auf Mobile nicht funktioniert verliert täglich Kunden. Ich designe zuerst für Mobile und skaliere dann auf Desktop hoch.
